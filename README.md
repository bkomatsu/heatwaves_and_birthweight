# The underlying mechanisms behind the heatwave impacts on birth outcomes: Evidence from a new birth cohort study in Brazil
This repository contains replication files for the article "The underlying mechanisms behind the heatwave impacts on birth outcomes: Evidence from a new birth cohort study in Brazil", by Komatsu and Menezes-Filho (2025). Please cite this paper as: 

```
Komatsu, B. K., Menezes-Filho, N. (2025). "The underlying mechanisms behind the heatwave impacts on birth outcomes: Evidence from a new birth cohort study in Brazil". Working paper.
```

## How to Replicate the Paper's Results
Do-file `plosone_analysis_registry_data.do` runs in Stata all results involving newborn registry data. We put large file in a zipped folder. To reproduce the results, please download this repository, unzip the compressed folder, change main folder and location in do-file and run it.

Links to original data sources are in file `data_sources.txt`
